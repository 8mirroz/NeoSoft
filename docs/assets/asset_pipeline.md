# Конвейер ассетов и политика загрузки ресурсов (Asset Pipeline) — Genesis v5.1

Для исключения сбоев на мобильных устройствах из-за отсутствующих файлов или поврежденных ресурсов, в проекте **Neo Soft Frost** все внешние медиа-файлы (текстуры, звуки, шрифты) загружаются строго через **Манифест ассетов** с обязательным использованием **fallback-заглушек**.

---

## 1. Манифест ассетов

Манифест представляет собой JSON-файл `data/assets/asset_manifest.json`, содержащий реестр путей к ресурсам, сгруппированных по категориям (gems, ui, audio, fonts). 

Логический код обращается к манифесту, чтобы получить локальный путь (`res://...`), перед тем как выполнить вызов `load()` или `preload()`.

---

## 2. Fallback-политика безопасной загрузки (Graceful Degradation)

Любая попытка прямой загрузки через `load("res://path")` запрещена. Все ресурсы должны загружаться через глобальный утилитарный класс-загрузчик с обработкой ошибок:

```gdscript
# Пример паттерна безопасной загрузки
static func safe_load_texture(category: String, key: String) -> Texture2D:
	var path = AssetManifest.get_path(category, key)
	if path == "" or not ResourceLoader.exists(path):
		printerr("Asset not found in manifest or filesystem: ", category, "/", key)
		return AssetManifest.get_fallback_texture(category)
	return load(path) as Texture2D
```

### Fallbacks по умолчанию:
* **Текстуры гемов**: Обычный белый круг с логом ошибки в консоли.
* **Звуковые эффекты**: Пустой аудио-поток без вывода звука и без краша микшера.
* **Шрифты**: Стандартный системный шрифт Godot (sans-serif).
* **Шейдеры**: Плоский полупрозрачный материал (вместо heavy blur).

---

## 3. Требования к оптимизации ассетов
1. **Текстуры**: Все спрайты гемов сжаты в WebP/PNG с включенным MIP-mapping для мобильных устройств.
2. **Spritesheets**: Игровые элементы и VFX сгруппированы в атласы (spritesheets) размером не более 2048x2048 для оптимизации видеопамяти.
3. **Аудио**: Звуки эффектов (SFX) в формате `.wav` с низкой задержкой, музыкальные петли (Ambient) — `.ogg`.

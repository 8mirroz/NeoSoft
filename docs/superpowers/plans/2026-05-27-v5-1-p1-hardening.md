# Neo Soft Frost v5.1 P1 Hardening Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** довести уже существующие P1 подсистемы до production-ready состояния: интеграция в runtime, детерминизм replay, баланс спец-сфер, тестовые гарантии производительности.

**Architecture:** ядро `core_match3` остается детерминированным и изолированным от UI; интеграция выполняется через `LevelSession` + `EventBus` + typed contracts. UI/feedback получают только события, без прямой мутации board state.

**Tech Stack:** Godot 4.5.x, GDScript, JSON configs (`data/*`), GUT (`addons/gut`), CI headless runner.

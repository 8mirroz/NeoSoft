# Project Structure — Genesis v5.1 Production

Этот документ фиксирует официальную структуру каталогов и правила размещения ресурсов для проекта **Neo Soft Frost**. Любое добавление новых файлов должно строго следовать этой структуре.

```text

│   ├── audio/                          # Аудиофайлы (SFX, музыкальные петли)

│   ├── particles/                      # Текстуры и настройки систем частиц
│   └── ui/                             # Графические ассеты интерфейса (рамки, иконки)
│
├── config/                             # Дополнительные системные конфигурации
│
├── data/                               # Внешние конфигурации баланса и экономики (JSON)
│   ├── assets/                         # Списки манифестов ресурсов
│   ├── economy/                        # Профили наград и покупок
│   ├── feedback/                       # Параметры сочности эффектов
```

## Правила именования (Naming Conventions)

1. **Директории**: Строго в нижнем регистре со змеиным регистром (`snake_case`). Например: `core_match3`.
2. **Скрипты**: Строго `snake_case`. Например: `resolve_pipeline.gd`.
3. **Сцены**: Строго `snake_case`. Например: `loading_screen.tscn`.
4. **Классы (`class_name`)**: Строго `PascalCase`. Например: `ResolvePipeline`.
5. **Сигналы / События**: Строго `snake_case`. Например: `match_detected`.

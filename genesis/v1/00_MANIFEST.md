# Genesis v1 - Version Manifest

**Дата**: 2026-05-26  
**Статус**: Активен  
**Предыдущая версия**: Отсутствует (Инициализация)  

## Цели версии
Подготовка и верификация стабильного, протестированного выпуска MVP игры **Neo Soft Frost** для внешнего тестирования пользователями, сбора обратной связи (CSAT-оценки), фикса критических визуальных расхождений (поддержка 8 типов фишек) и устранения рассинхронизации ввода (блокировка тапов во время анимаций).

## Ключевые изменения
- **Внедрение Input Blocker (R1)**: Игнорирование любых кликов и жестов игрока во время активных анимаций обвала и спавна гемов.
- **Поддержка 8 видов гемов (S1)**: Расширение `board_view.gd` для полной визуальной симметрии с `gem_view.gd` (устранение дрейфа).
- **Сбор CSAT-отзывов**: Интеграция формы обратной связи с оценкой 1–5 звезд и полем ввода на экран результатов уровня.
- **Локальный экспортер логов**: Добавление кнопки копирования/экспорта аналитических логов и CSAT-оценок в меню настроек для легкой отправки результатов тестерами.
- **Переключатель качества**: Управление качеством (`web_default` vs `android_safe`) прямо из меню настроек.

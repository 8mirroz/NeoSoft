# ADR 005: Интеллектуальный выбор целей (Target Priority System)

## Статус
Принято (Active)

## Контекст
Наводящиеся сферы (созданные из квадратов 2x2) должны автоматически лететь во "вкусные" или "полезные" ячейки. Если наведение будет полностью случайным, игрок почувствует разочарование от неэффективного использования спец-элемента. Требуется интеллектуальный алгоритм ранжирования целей на основе текущего состояния и задач уровня.

## Решение
Внедрить подсистему `TargetPrioritySystem`, которая возвращает оптимальную ячейку-мишень для наводящихся эффектов на основе ранжированного списка приоритетов:

1. **Матрица приоритетов**:
   - `Приоритет 1: Критические цели уровня` (Boss-объекты, Древние ядра, порталы энергии), уничтожение которых напрямую ведет к победе.
   - `Приоритет 2: Заблокированные цели уровня` (ячейки целей, находящиеся под льдом, камнем, цепями).
   - `Приоритет 3: Труднодоступные цели` (угловые ячейки, узкие проходы, нижние ряды поля).
   - `Приоритет 4: Препятствия с низким запасом здоровья (HP)` (добивание blocker-элементов).
   - `Приоритет 5: Стратегические препятствия` (препятствия с высоким уровнем HP, мешающие осыпанию сфер).
   - `Приоритет 6: Случайная полезная ячейка` (сфера цвета, необходимого для сбора по целям уровня, или обычный матч).

2. **Рейтинг ценности ячеек**:
   Каждой ячейке поля на основе сканирования присваивается числовой рейтинг:
   $$Score(Cell) = W_{type} \times Mult_{state} + Bonus_{location}$$
   Сфера гарантированно летит в ячейку с максимальным рейтингом.

3. **Событийное кеширование**:
   Список целей обновляется в начале каждого `ResolvePipeline` и кешируется, предотвращая повторные тяжелые сканирования поля при активации нескольких наводящихся сфер подряд.

## Последствия
- Высокое удовлетворение игрока — наводящиеся сферы действительно помогают проходить сложные уровни.
- Простая калибровка ИИ-поведения под разные типы уровней.
- Необходимость синхронизации целей уровня с логическим ядром (контроллер должен передавать текущие условия победы в `TargetPrioritySystem`).

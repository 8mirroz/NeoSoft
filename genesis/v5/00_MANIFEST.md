## Version Goals
Переход от Combo Fever Engine (v4) к полноценному **Controlled Cascade Pleasure Engine** — production-ready модулю, объединяющему:
- Shape Merge Matrix 2.1 (11 типов форм → 10 типов спец-сфер)
- Controlled Cascade Engine (Natural / Assisted / Cinematic Drop)
- Balance Governor (anti-fake, anti-auto-win, cascade depth limits)
- Special Combo Matrix (15 комбинаций power-up + power-up)

- Fever Meter + Fever Rain
- Target Priority System 2.0

- Telemetry + Auto-Balancing Layer
- **UI/UX System** — 10 экранов game screens (Loading, Main Menu, World Map, Level Preview, Gameplay, Pause, Level Complete, Out of Moves, Daily Rewards, Shop)

# Genesis v3 - Version Manifest

**Date**: 2026-05-26
**Status**: Active
**Previous**: v2

## Version Goals
Полная переработка главного меню игры Neo Soft Frost в соответствии с референсным дизайном "Neo Soft Frost" (пастельный glassmorphism, радужная сфера, 4-кнопочная навигация, верхний бар валют, нижний бар навигации).

## Major Changes
- Полный редизайн главного меню по новому референсу
- Добавление верхней панели валют (Coins + Stars + Inbox)
- Добавление центральной декоративной радужной сферы
- 4-кнопочная сетка (Levels, Events, Shop, Settings)
- Нижняя навигационная панель (Home, Rankings, Collection, Friends, Inbox)
- Переход цветовой палитры с тёмной на пастельно-лавандовую
- Пастельный glassmorphism вместо тёмного glassmorphism

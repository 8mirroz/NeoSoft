# ADR 002: Переход на пастельную палитру Glassmorphism

*   **Статус**: Утверждено
*   **Дата**: 2026-05-26
*   **Контекст**: Текущее меню (v2) использует тёмную палитру (`Color(0.04, 0.03, 0.07)` фон, белые полупрозрачные рамки). Референс требует кардинальной смены на пастельно-лавандовую палитру с полупрозрачными белыми стеклянными элементами.

---

## Варианты решения

1. **Параметрический theme-switcher**: Создать систему themes, позволяющую переключаться между тёмной и светлой палитрами.
   - *Минусы*: Избыточная сложность для текущей задачи. Два divergent стиля потребуют удвоения дизайн-токенов.

2. **Прямая замена цветовых констант**: Заменить все цвета в `_apply_theme()` и `_draw()` на пастельную палитру.
   - *Плюсы*: Минимальная сложность, всё в одном файле, идеально для процедурного рендеринга.

---

## Решение

**Выбран Вариант 2.**

### Палитра (Design Tokens)

| Токен | Назначение | Цвет |
|---|---|---|
| `BG_TOP` | Верх градиента фона | `Color(0.92, 0.88, 0.98, 1.0)` |
| `BG_BOTTOM` | Низ градиента фона | `Color(0.96, 0.90, 0.95, 1.0)` |
| `GLASS_BG` | Фон glassmorphic панелей | `Color(1.0, 1.0, 1.0, 0.25)` |
| `GLASS_BORDER` | Рамка glassmorphic панелей | `Color(1.0, 1.0, 1.0, 0.55)` |
| `TEXT_PRIMARY` | Основной текст | `Color(0.55, 0.48, 0.65, 1.0)` |
| `TEXT_SECONDARY` | Вторичный текст | `Color(0.65, 0.58, 0.72, 0.85)` |
| `ACCENT_PINK` | Розовый акцент | `Color(0.95, 0.75, 0.85, 1.0)` |
| `ACCENT_BLUE` | Голубой акцент | `Color(0.75, 0.85, 0.98, 1.0)` |
| `ACCENT_LAVENDER` | Лавандовый акцент | `Color(0.82, 0.75, 0.95, 1.0)` |
| `GLOW_BUBBLE` | Цвет пузырей | `Color(0.90, 0.85, 0.98, 0.15)` |
| `PLAY_GRADIENT_L` | Левый край кнопки Play | `Color(0.92, 0.82, 0.95, 0.55)` |
| `PLAY_GRADIENT_R` | Правый край кнопки Play | `Color(0.82, 0.88, 0.98, 0.55)` |
| `NAV_ACTIVE` | Подсветка активной вкладки | `Color(0.90, 0.72, 0.85, 1.0)` |
| `NAV_INACTIVE` | Неактивная вкладка | `Color(0.65, 0.60, 0.72, 0.7)` |

---

## Последствия

- Полная визуальная трансформация меню при нулевой архитектурной перестройке.
- Стеклянные элементы (white alpha 0.25-0.55) обеспечивают читаемость на пастельном фоне.
- Тени меняются с чёрных на мягкие лавандовые.

# ADR 001: Сохранение Tech Stack из v2

*   **Статус**: Утверждено
*   **Дата**: 2026-05-26
*   **Контекст**: Главное меню v3 не требует изменений технического стека. Проект продолжает использовать Godot 4.5, GDScript, процедурную отрисовку `_draw()`, `CPUParticles2D`, `StyleBoxFlat` для glassmorphism.

## Решение

Tech stack полностью наследуется из `genesis/v2/03_ADR/ADR_001_TECH_STACK.md`. Изменений нет.

## Последствия

Нулевой риск миграции. Все компоненты совместимы.

# Genesis v2 - Version Manifest

**Дата**: 2026-05-26  
**Статус**: Активен  
**Предыдущая версия**: v1 (MVP Release Candidate)  

## Цели версии
Перевод визуального качества, отзывчивости и игрового опыта (UI/UX) игры **Neo Soft Frost** на премиальный мировой уровень. Это включает внедрение десинхронизированного дыхания сфер на поле, запуск пошаговых цепных комбо-каскадов с паузами, тряской и звуковыми акцентами, проработку уникальных VFX взрывов для каждого типа сфер и замену главного меню на интерактивное лобби с глубоким параллаксом слоев и глассморфизмом.

## Ключевые изменения
- **Пошаговый Cascade Sequencer (Combo-007)**: Внедрение асинхронной очереди визуальных событий `VisualEventQueue` в `BoardView` для последовательной отработки взрывов, падений и Refill-спавнов с театральными паузами.
- **Иммерсивные комбо-акценты (Combo-008)**: Реализация тряски поля, всплывающих неоновых комбо-лейблов и прогрессивного повышения тона звуков (pitch shift) при цепных комбо.
- **Высококонтрастные анимированные сферы (Visual-006)**: Улучшение процедурной контрастности контуров и бликов, а также разработка 8 уникальных idle-анимаций (синусоидальное вращение роз, дыхание волн, подъем пузырьков).
- **Стихийные VFX взрывы сфер (VFX-009)**: Процедурный спавн индивидуальных частиц `CPUParticles2D` для каждого из 8 видов гемов (розовые лепестки, синие брызги, ледяные осколки, мятные вспышки).
- **Премиум-лобби с параллаксом (Menu-010)**: Интерактивный трехслойный параллакс фона в зависимости от мыши, стеклянные адаптивные панели с размытием и микро-анимации плавной пульсации кнопок при наведении.

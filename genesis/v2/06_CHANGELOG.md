# Changelog - Genesis v2

> Этот файл фиксирует изменения в рамках итерации версии v2 (управляется с помощью `/change`). Большие новые фичи требуют создания новой архитектурной ветки через `/genesis`.

## Формат лога
- **[ADD]** Добавление новых разделов или архитектурных требований
- **[CHANGE]** Корректировка существующих требований
- **[FIX]** Исправление ошибок проектирования
